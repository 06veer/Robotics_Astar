Abstract
--------

This project implements an Autonomous Robot Path Planning and Obstacle Avoidance simulator using the A* algorithm. The simulator presents the environment as a 2.5D isometric grid with orbit controls, allowing interactive placement of obstacles, start and goal nodes, and visualizing the A* search process including open and closed sets, and the final shortest path. A simulated robot animates along the computed path demonstrating obstacle avoidance and path-following behavior.

The project is useful as a teaching and demonstration tool for robotics path planning, search algorithms, and obstacle avoidance in environments such as labs, warehouses, and other grid-based navigation scenarios.
